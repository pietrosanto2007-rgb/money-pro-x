# Money Pro X (Next.js)

This project is an in-progress migration of the original single-file prototype into a Next.js app.

## Dev

```bash
npm install
npm run dev
```

## Notes

- The legacy UI markup lives in `src/legacy/body.html`.
- The legacy logic is still vanilla JS in `public/legacy/app.js` and runs client-side.
- The original source is preserved in `src/legacy/index.html`.
